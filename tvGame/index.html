Разработано для курсового проекта
Данное приложение предсталяет из себя набор игр.
Меню
В меню можно выбрать игру. ¯\_(ツ)_/¯
Змея
Данная игра представляет из себя класическую змейку. В каждой игре сохраняется личный рекорд, который можно обнулить через пульт ДУ справа на экране.
Птица
Моя любимая игра из всех, написанная с помощью канваса. Кстати у каждой игры свой прекрасный саундтрек (все тот же пульт ДУ:))
Понг
Сам по себе понг расчитан на игру вдвоем, но в моем понге нужно как можно дольше продержаться против компьютера, имея 5 жизней.
